# Fedivership

## これはなに?
ActivityPubに対応した分散型SNS用のクライアントです。

## 使い方
分散型SNSのインスタンスアドレスを入力してログインすれば使えます。

## 対応OS
Windows10 64bit
* Mac,Linuxについてはそのうち対応版を作ります。(たぶん)
